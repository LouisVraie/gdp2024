# Gestion de projet TD noté

## État des version

- `v1` : Début du projet
- `v2` : Version aboutie
- `v3` : N'est pas laçable
- `v4` : Version incomplète des attendus

## Lancer le projet
```bash
docker-compose up -d --build
```

ou (sur Windows)

```bash
.\Start.cmd 
```
